# Flowsave 

> A clean, aesthetic Instagram reel & image downloader — built with love by [SARTHIK](https://github.com/SARTHIKcodes)

---

##  Project Description

Flowsave is a minimal and aesthetic web app that lets you download Instagram reels and images with ease. Just paste an Instagram link, hit download, and the media opens in a new tab where you can save it directly to your device. No watermarks, no sign-in required.

---

## ✨ Features

-  Paste any Instagram reel or post link and download instantly
-  Clean dark aesthetic UI with anime mascot
-  API key is hidden securely on the backend — never exposed to the browser
-  Fast response powered by RapidAPI
-  Runs locally with a simple Node.js server
